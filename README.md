# ChatPetLovers

Chatbot interativo para um PetShop. Nesse projeto, o cliente é capaz de solicitar o serviço de banho e tosa e serviço de busca e entrega do pet.

Fluxo: o chatbot inicia o atendimento e o usuário realiza até 5 interações. O sistema responde com base no contexto definido

Execução: Configure sua API Key do Gemini no ambiente (Google Colab ou local) -> Execute o script Python -> Interaja com o chatbot via terminal.

Ao final:

Um resumo da conversa é gerado Os dados são estruturados em JSON
